# WP Top 5 Pro

WP Top 5 Pro is a WordPress plugin that allows you to extract and display the top 5 key points from your content. It integrates with OpenAI to generate the key points and offers various display options such as above, below, or in a popup.

## Features

- Extract top 5 key points from your content using OpenAI.
- Display the key points in different styles: above content, below content, or in a popup.
- Light and dark display modes.
- Easy-to-use shortcode with customizable attributes.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/wp-top-5-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> WP Top 5 Pro screen to configure the plugin.

## Usage

To display the top 5 key points, use the `[wp_top_5]` shortcode in your content. You can customize the view, position, and mode using various attributes.
